# AS Theme

**AS Theme** is a sleek, modern Visual Studio Code theme designed for clarity, focus, and long coding hours — with a subtle, dark *Stranger Things*-inspired atmosphere that sparks creativity and mystery.

---

## Features

- Dark mode with rich, deep colors and high contrast  
- Carefully chosen color palette to reduce eye strain  
- Clean syntax highlighting for all major programming languages  
- Support for UI and syntax elements (comments, keywords, strings, variables, etc.)  
- Inspired by the moody, mysterious vibe of *Stranger Things* — perfect for night owls and creative coders

## Installation

You can install **AS Theme** directly from the Visual Studio Marketplace:

1. Open Visual Studio Code  
2. Go to Extensions (`Ctrl+Shift+X` or `Cmd+Shift+X`)  
3. Search for `AS Theme` by `ArpitTheme`  
4. Click **Install**  
5. Go to **Color Theme** picker (`Ctrl+K Ctrl+T`) and select **AS Theme**

---

## Usage

Once installed, activate the theme via:

- Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`) → `Preferences: Color Theme` → select **AS Theme**
